# Language Rules

Use the language selected during `/setup-ralph-matsuo` for all ongoing assistant responses in this repository.

- Scope: conversations and interactive assistant replies
- Keep variable names, function names, type names, file names, package keys, and other conventional code identifiers in English
- Leave fields that are conventionally English, such as `package.json` descriptions, in English unless the repository explicitly adopts another rule
- If no language has been selected yet, ask the user which language to use and then update this file
