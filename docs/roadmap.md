# Roadmap

Use this document for repo-level direction. Detailed execution status belongs in each PRD's `progress.md`.

## Current Focus

- Keep the repository reusable as an OSS template
- Stabilize the contract between Claude Code, Ralph Loop, and GitHub Actions
- Make setup and documentation clearer for adopters

## Active PRDs

This template repository does not keep product-specific PRDs checked in by default.
Create `docs/prds/prd-{slug}/` when planning work for the template itself or when adopting Ralph in another repository.

## Future Ideas

- Localize the onboarding guide into additional languages
- Add more repository health checks
- Provide setup examples for non-npm repositories
- Improve headless execution observability
